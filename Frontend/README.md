# 🎓 Web Education Platform

## 📌 Description
This is a full-stack web application built for education purposes.

The system allows users to:
- Register & login
- Manage courses
- Submit exercises
- View results

## 🚀 Technologies Used
- Frontend: React (Vite)
- Backend: Node.js (Express)
- Database: MySQL

## ⚙️ Installation & Run

### 1. Clone project
git clone https://github.com/KhanhLam00/Wed.edu.git

### 2. Backend
cd Backend
npm install
npm start

### 3. Frontend
cd Frontend
npm install
npm run dev
